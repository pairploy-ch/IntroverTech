Website Scraping
รับดึงข้อมูล รวบรวมข้อมูลจากเว็บไซด์ หรือฐาน ข้อมูลอื่นๆ ที่ต้องการ

รับดึงข้อมูลจากเว็บไซด์ หรือฐานข้อมูลอื่นๆ ตามความต้องการของลูกค้าโดยข้อมูลที่จะได้จะอยู่ในรูปแบบของไฟล์ word, excel, csv, txt, sql เป็นต้นลูกค้าสามารถแจ้งรายละเอียดเข้ามาคุยกันก่อนได้เลยครับ เดี๋ยวลองทำตัวอย่างให้ดูก่อนได้ครับเน้นคุณภาพ ไม่จ้างไม่ว่ากันครับ (ราคาประเมินกันทีหลังได้)

หลักๆ ผมจะใช้ Python ในการดึงข้อมูลให้ครับสามารถเก็บข้อมูลได้เกือบทุกอย่าง ไม่ว่าจะเป็นข้อความ ตัวเลข ภาพ link หรือโหลดไฟล์ออกมาเก็บไว้ใน folder ที่ลูกค้าต้องการ

ตัวอย่างการดึงข้อมูล เช่น
- ราคา คริปโต, forex,ทอง น้ำมัน (รวมถึงการ filter scan เผื่อหาราคาให้ตรงกับแผนการเทรดด้วย Indicator)
- ราคาสินค้าต่าง- การดึงข้อมูลจาก post ต่างๆ จาก facebook
- การดึงข้อมูลรายการสินค้า และรายละเอียดของสินค้าจากร้านค้าออนไลน์ เช่น Alibaba.com
- การเช็คยอดเงิน- อื่นๆ ตามที่ลูกค้าต้องการ
หมายเหตุ : ระยะเวลา และราคา แล้วแต่ความยากง่ายของงาน

ขั้นตอนการทำงาน
1. แจ้ง url ของ website ที่ต้องการจะดึงข้อมูล
2. แจ้งความต้องการของลูกค้า
3. ประเมินราคา และระยะเวลาของงาน
4. อัปเดตงาน
5. ส่งมอบงาน
